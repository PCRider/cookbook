==========
Rust Notes
==========

Option 1) Install with rustup::

  curl https://sh.rustup.rs -sSf | sh
  # Don't forget to update the PATH (~/.cargo/bin)

Option 2) Installing compiler with apt::

  sudo apt-get install rustc

Start a new project::

  cargo new ~/myproject

Run a project::

  # From the project directory
  cargo run

Get docs with::

  rustup doc
